📖 Documentation

Quick Start

Get up and running in 5 minutes with IEEE LaTeX Editor.

  1. Load a Document: Click New to start with template, or paste existing LaTeX code
  2. Write Content: Edit LaTeX code on the left side
  3. Compile & Preview: Click Compile or press Ctrl+Enter
  4. Save Your Work: Use Save or Ctrl+S to download .tex file
  5. Export: Click Export to save rendered paper as HTML

Interface Overview

Top Menu Bar: File operations, text formatting, structure, and math tools

View Modes:

  • ⊟ Split - Code editor left, preview right (default)
  • ⌨ Code - Full-width LaTeX editor
  • ◉ Preview - Full-width rendered paper
  • ✎ Edit - Word processor-style editing

Right Sidebar: Document outline for navigation and quick snippets

Status Bar: Shows compilation status, cursor position, and document stats

Editing LaTeX Code

Auto-Compilation: Document compiles automatically after you stop typing (configurable in Settings)

\textbf{bold} \textit{italic} $math equation$ \section{Title} \cite{ref}

Find & Replace: Press Ctrl+F to search the LaTeX code

Line Numbers: Toggle in Settings for cleaner view

WYSIWYG Editing Mode

Tips & Tricks

  • Use Snippets library for common elements like figures and tables
  • Switch between split, code, and preview views as needed
  • Use document outline sidebar for quick navigation
  • WYSIWYG mode is great for final proofreading
  • Auto-save stores changes to browser storage
  • Adjust auto-compile delay in Settings for better performance
  • Press Find button or Ctrl+F to search LaTeX code
  • Each theme (Dark/Darker/Light) affects only the UI, not the paper
